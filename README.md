## Backend
### To install backend dependencies:
```npm install express dotenv mysql2```

### To use dotenv.config():
1. `cd back_end/`
2. Create `.env` file locally.
3. Add the following to `.env`.
```
MYSQL_HOST="<ip_addr>"
MYSQL_USER="<username>"
MYSQL_PASSWORD="<password>"
MYSQL_DATABASE="<db_name>"
```
